Railway Reservation System










    This is a menu-driven Railway Reservation System developed using C programming. The system uses file handling to manage train details and ticket reservations.








Objective






To design and implement a Railway Reservation System that allows users to view trains, book tickets, cancel tickets, and manage reservation records efficiently.




Features




Add train details
Display available trains
Book ticket
Cancel ticket
File-based data storage


Technologies Used




C Programming
File Handling




Project Type


Individual Project
